# Welcome to VeriFace!
## Introduction
Veriface is an AI assisted check-in system for classrooms and social events.
## Tech Stack
This project use the following technology:  
* Python 3.12
* MySQL
* Amazon Web Service for hosting API and DB
* Visual Studio Code for IDE
* FastAPI
